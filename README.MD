# Replorer
Утилита для быстрого перезапуска Windows Explorer помогает в случае проблем и глюков с миниатюрами фотографий, видеофайлов и иконок программ.

Жёстко перезапускает **Windows Explorer** и мгновенно восстанавливает ранее открытые папки.

## Что делает
- Сохраняет список **реально открытых** окон Проводника (только каталоги файловой системы).
- Принудительно завершает все процессы `explorer.exe`.
- **Не** запускает Проводник вручную — полагается на его автоперезапуск системой.
- Сразу после завершения запускает восстановление: открывает те же папки новыми окнами.

> Виртуальные окна («Быстрый доступ», «Этот компьютер», «Домой» и т. п.) намеренно **игнорируются** — восстанавливаются только реальные пути.

## Зачем это нужно
- Лечит «залипшие» иконки, отвалившиеся миниатюры, глюки панели задач и Проводника.
- Быстрый reset оболочки без выхода из системы и без перезагрузки.

## Использование
1. Расположите файл `Replorer.exe` в удобное место, например на рабочем столе.
2. Закройте несохранённые документы в сторонних приложениях (по желанию).
3. Запустите файл. Проводник будет завершён и автоматически поднят системой.

> Если перед перезапуском не было открытых папок, Replorer ничего не открывает.

## Поддержка
- Windows 10/11 (x64/x86).
- Права администратора **обычно не требуются**.
- В редких корпоративных или «твикнутых» конфигурациях автоперезапуск Проводника может быть отключён — тогда откройте его вручную (Win+R → `explorer`).

## Ограничения и примечания
- Восстанавливаются только окна с путями на диске. Виртуальные представления не открываются.
- Завершение `explorer.exe` не влияет на сторонние приложения, но может прервать операции копирования/перемещения, запущенные в Проводнике.
- Утилита без консоли и без инсталляции, один файл `Replorer.exe`.
